**Практическое задание**  

Цель задания: научиться создавать Web-приложения машинного обучения.  

**Задание:** 

1)Сформируйте команду из 3-4 человек. 

2)Создайте репозиторий на GitHub, который будет 
использоваться для командной разработки Web-приложения. 

3)Выберите предварительно обученную модель машинного обучения, 
которую вы хотите использовать в своем приложении.  

4)Разработайте Web-приложение на основе библиотеки Streamlit, 
которое использует выбранную вами модель. 

5)Опубликуйте код Web-приложения в репозитории на GitHub.  

6)Отправьте ссылку на созданный репозиторий 
GitHub на платформу для проверки.
